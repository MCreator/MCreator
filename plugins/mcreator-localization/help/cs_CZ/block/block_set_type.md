Tento parametr ovládá zvuky a chování některých základen bloků:

- **Tlačítko**: Dřevěná tlačítka mohou být zmáčknuta šípy a zůstávají stisknuta déle.
- **Plot**: Dřevěné ploty a ploty z jiného materiálu se vzájemně nespojují.
- **Nášlapná deska**: dřevěné nášlapné desky mohou být zašlápnuty jakoukoliv entitou, nejen moby a hráči.
- **Dveře/Padací dveře**: Železné dveře a padací dveře nelze otevřít ručně.

Quando este procedimento retornar verdadeiro, a reprodução da animação será iniciada, a menos que a animação já esteja em execução.

Quando este procedimento retornar falso, a reprodução da animação será interrompida.

Se a troca acontecer muito rápido, pode não haver tempo suficiente para a animação iniciar ou reiniciar a reprodução.

Se a animação não estiver configurada para repetir (loop), ela será interrompida ao chegar ao final, e a condição precisará alternar de true para false e novamente para true para reiniciar a animação.

Para animações de caminhada, este parâmetro controla quando a animação será reproduzida.

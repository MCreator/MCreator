Dieser Parameter steuert den Namen dieses Puzzle-Pools, der vom aktuellen als Fallback verwendet werden kann, falls er nicht an der angegebenen Position generiert werden kann.

Wenn "Zielpool" von Puzzle-Blöcken in deinen Teilen auf `${modid}:${registryname}_<pool_name>` gesetzt ist, wird das Spiel Teile aus diesem Pool gegen sie platzieren.

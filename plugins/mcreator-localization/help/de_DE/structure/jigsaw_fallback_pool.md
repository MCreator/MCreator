Dieser Parameter legt fest, welcher Puzzlepool als Ausweichlösung dient, um ein „Terminator“-Puzzleteil auszuwählen,
falls in dem Fall, dass kein Teil aus diesem Pool an der angegebenen Position generiert werden kann (nicht genügend Platz oder Tiefenbegrenzung erreicht), die passende Komponente ausgewählt wird.

Wenn auf `${modid}:${registryname}_<pool_name>`gesetzt, wird das Spiel Puzzleteile aus Pool `<pool_name>` in dieser Liste auswählen.

Der gleiche Pool kann ebenfalls genutzt werden, aber wenn er sich mit anderen Puzzleteilen derselben Struktur überschneidet, wird er ignoriert.

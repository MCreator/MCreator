Wenn aktiviert, wird die Struktur im angegebenen Höhenbereich gestartet.
Wenn dieser Parameter aktiviert ist, wird die Bodenerkennung deaktiviert und die
Struktur wird stattdessen im angegebenen Höhenbereich platziert.

Du kannst verschiedene Höhenbereichsverteilungen wählen, je nachdem, wie du die
Chance auf die Verteilung der Struktur in einer bestimmten Höhe haben möchtest.

Hier kannst du einstellen, wie oft die Entität in jeder Überfalls-Welle spawnen wird.

Abhängig von der Schwierigkeit wird es eine gewisse Anzahl von Wellen geben:

- Friedlich: 0
- Einfach: 3
- Normal: 5
- Schwer: 7

Die Anzahl der Entitäten, die pro Überfalls-Welle generiert werden, kann eingestellt werden, wenn die Entität ein Überfall-Typ ist.

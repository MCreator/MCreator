Diese Nachricht wird im Chat angezeigt, wenn ein Spieler von einer Kreatur mit benanntem Gegenstand getötet wurde.

Du kannst die folgenden Token in dieser Nachricht benutzen:

- '<player>': der Name des getöteten Spielers
- '<attacker>': der Name der Kreatur, die den Schaden zufügte
- '<item>': der Name des Gegenstands, der vom Angreifer gehalten wurde

Verwende diesen Auslöser, um zusätzliche Werkzeuge mit Drop/Erntezustand anzugeben.
Sowohl Vanilla als auch benutzerdefinierte Erntebedingungen müssen erfüllt werden, damit der Block mit diesem Tool geerntet werden kann.

Wenn diese Prozedur falsch zurückgibt, wird das Werkzeug den Block-Drop nicht abholen.

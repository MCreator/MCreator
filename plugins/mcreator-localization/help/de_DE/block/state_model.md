Wählen Sie das Modell aus, das mit diesem Blockstatus verwendet werden soll. Das Modell definiert nur das visuelle Erscheinungsbild und nicht die
Begrenzungsbox des Blocks.

- **Normal** – normaler Block mit Texturen auf jeder Seite
- Benutzerdefiniert – Du kannst auch benutzerdefinierte JSON- und OBJ-Modelle definieren

Bei der Erstellung benutzerdefinierter Modelle wird JSON aufgrund der Vanilla Unterstützung für diesen Modelltyp empfohlen.

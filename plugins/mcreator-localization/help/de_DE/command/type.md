Mit dieser Option kannst du den Typ des Befehls festlegen und wie/wo er verwendet werden kann.

* **Standard**: Der Befehl kann überall verwendet werden. Dies ist das übliche/normale Verhalten (z.B. `/give`).
* **Nur Einzelspieler**: Der Befehl wird nur auf Singleplayer-Welten verfügbar sein (z.B. `/publish`).
* **Nur Mehrspieler**: Der Befehl wird nur auf Mehrspieler-Servern verfügbar sein (z.B. `/ban`).
* **Clientseitig**: Der Befehl wird ausschließlich auf der Client-Seite sein, was bedeutet, dass nur Aktionen auf dem Client ausgeführt werden können.

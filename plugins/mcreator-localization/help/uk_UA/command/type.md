Цей параметр дозволяє визначити тип команди й те, як/де вона може бути використана.

* **Стандартний**: команду можна використовувати будь-де. Це звичайна/нормальна поведінка (наприклад, `/give`).
* **Гра наодинці**: команда буде доступна лише у світах гри наодинці (наприклад, `/publish`).
* **Гра в мережі**: команда буде доступна лише на серверах (наприклад, `/ban`).
* **Клієнтський**: команду можна буде виконувати виключно на стороні клієнта, тобто виконуватимуться лише дії на стороні клієнта.

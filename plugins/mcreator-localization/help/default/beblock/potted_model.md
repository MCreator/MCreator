This allows to change the block's model when it is inside a flower pot.

* **Normal** - This is the model you selected for your regular block
* Cross - Block with textures in an X shape like flowers.
* Full cube - This is the regular 3D Minecraft cube model
* Custom - you can define custom Bedrock (`.geo.json` files) models too. Your block is limited to 30×30×30 pixels in size.

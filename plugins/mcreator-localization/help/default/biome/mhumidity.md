<h2>
Humidity
</h2>
<ul>
    <li>Controls the biome placement.</li>
    <li>Example: 
        <ul>
            <li>Desert biome generates from -1 to -0.35</li>
            <li>Bamboo jungle biome generates from 0.3 to 1</li>
        </ul>
    </li>
    <li>First value must be <b>lower</b> than second one</li>
</ul>

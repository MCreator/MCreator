All generators support an M.m.p (Major, minor, patch) format where the number of segments can be increased or decreased.
<br/><br/>
Most generators (all except resourcepack, datapack, and addon) support using a modified version of Semantic Versioning.
<br/><br/>
SemVer allows mods to have suffixes in their versions. These can be used to show development stage or Minecraft version. Examples include:
- 3.5.2f
- 2.0-beta1
- 3.0.0rc
- 1.0+1.21.1
- 2.2.0-snapshot+mc1.21.8

All generators support an M.m.p (Major, minor, patch) format where the number of segments can be increased or decreased.

All generators except the resourcepack and datapack generators support using a modified version of SemVer (Semantic Versioning). 

SemVer allows mods to have suffixes in their versions. These can be used to convey a stage of development or the Minecraft version. Examples include:
- 3.5.2f
- 2.0-beta1
- 3.0.0rc
- 1.0+1.21.1
- 2.2.0-snapshot+mc1.21.8

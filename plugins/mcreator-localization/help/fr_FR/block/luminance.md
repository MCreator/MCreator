Ce paramètre contrôle la quantité de lumière émise par le bloc.

C'est une valeur comprise entre 0 et 15.

Si réglé sur 0, le bloc n'émettra pas de lumière. Si réglé sur 15, le bloc émettra autant de lumière que la pierre lumineuse.

Si contrôlé par une procédure (supporté par les éléments de mod bloc), assurez-vous de n'obtenir que les valeurs d'état de bloc et de ne pas les modifier, ou le jeu plantera.

De plus, gardez à l'esprit que la valeur est mise en cache, vous ne pouvez donc pas utiliser du hasard comme source de niveau de lumière.

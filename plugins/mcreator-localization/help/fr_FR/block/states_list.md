Cette liste définie les états de bloc pour ce bloc et leurs conditions.

Grâce à ce système, vous pouvez créer des blocs qui changent en fonction des
valeurs de leurs propriétés d'état de bloc.

Si une propriété d'état de bloc donnée est utilisée dans une condition d'état, elle doit être utilisée dans
toutes les conditions d'état.

Si la rotation de bloc est activée, MCreator générera automatiquement toutes les combinaisons d'états nécessaires pour gérer les rotations.

Les combinaisons de propriétés manquantes seront automatiquement remplies pour vous avec l'état par défaut du bloc.

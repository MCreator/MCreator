Ce paramètre détermine dans quel groupe d'échanges cette offre peut apparaître :

- **Commun** : échanges ayant plus de chances d'apparaître
- **Peu commun** : échanges ayant moins de chances d'apparaître
- **Achat** : échanges dans lesquels le marchand ambulant achète des objets, utilisé dans les versions 26.1 et supérieures de Minecraft

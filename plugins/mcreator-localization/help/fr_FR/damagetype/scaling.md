Ce paramètre détermine si les dégâts infligés doivent augmenter avec la difficulté.

- `Jamais` : la difficulté n'affecte pas les dégâts infligés
- `Toujours` : la difficulté augmente toujours les dégâts infligés, quelle qu'en soit la source
- `Lorsqu'ils sont causés par une entité vivante (non-joueur)` : la difficulté augmente les dégâts infligés uniquement s'ils sont causés par une entité vivante
  (comme les zombies ou les araignées).

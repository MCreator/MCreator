Si une entité est sélectionnée, un clic droit avec cet item fera apparaître l'entité dans le monde comme n'importe quel objet bloc. Le distributeur pourra également faire apparaître l'entité sélectionnée.
Sinon, la fonctionnalité sera désactivée.

Si un bloc à placer est également sélectionné, cette entité apparaîtra avant le bloc.

Lors de l'utilisation en mode survie, l'objet sera consommé.

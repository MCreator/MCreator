Si un bloc est sélectionné, un clic droit avec cet objet placera le bloc dans le monde comme
n'importe quel bloc. Sinon, la fonctionnalité sera désactivée.

Si une entité à placer est également sélectionnée, le placement de l'entité sera déclenché avant celui de ce bloc.

Lorsqu'il est utilisé en mode Survie, l'objet sera consommé.

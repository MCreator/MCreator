此參數決定這個村民在擁有這一職業後是否還會戴上由他們的類型定義的帽子:
* **None:** 無論如何村民都不會戴帽子
* **Partial:** 帽子不會完全覆蓋村民的頭部
* **Full:** 帽子會完全覆蓋村民的頭部

下表是TH相關的帽子（其中TH是類型材質決定的帽子，PH是職業材質決定的帽子）:

| TH 是否顯示      | TH = None | TH = Partial | TH = Full |
| ------------ |:---------:|:------------:|:---------:|
| PH = None    |    顯示     |      顯示      |    顯示     |
| PH = Partial |    顯示     |      顯示      |    隱藏     |
| PH = Full    |    隱藏     |      隱藏      |    隱藏     |

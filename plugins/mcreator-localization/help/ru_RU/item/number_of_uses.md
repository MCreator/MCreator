Приравняйте это значение к 0 для неограниченного использования данного предмета.

* Незеритовых инструментов: 2031 раз;
* Удочки: 64 раз;
* Огнива: 64 раз;

* Ножниц: 238 раз;
* Трезубца: 250 раз;

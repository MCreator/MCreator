Укажите значение 0 для неограниченного использования данного предмета.

* Алмазных — 1561 раз;
* Незеритовых — 2031 раз;
* Удочки — 64 раза;
* Огнива — 64 раза;
* Удочки с морковкой — 25 раз;
* Ножниц — 238 раз;
* Трезубца — 250 раз;

Selecione o modelo a ser usado com este estado de bloco. O modelo define apenas a aparência visual e não a caixa delimitadora do bloco.

- **Normal** - Bloco normal com texturas em cada lado
- Personalizado - você também pode definir modelos JSON e OBJ personalizados.

Ao fazer modelos personalizados, é recomendado um JSON devido ao suporte da vanilla para este tipo de modelo.
